About FORAS

FORAS is a tool to predict some available index in Yahoo Finance. The forecasting tool is powered using Deep Learning and expected to predict the future exchange rate/price as accurate as possible.

Currently there are 5 available index to be forecasted by the tool. You can choose the type of index on the top navigation bar.

Please note that we will not be responsible if the user experiencing any loss from using this tool. This tool will not determine the exchange rate/price from fundamental analysis. Please be careful when you decide to buy/sell based on this tool.
